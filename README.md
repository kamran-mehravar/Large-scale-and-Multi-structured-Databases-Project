# MehrCar

A Java and MongoDB project for large-scale and multi-structured databases, designed as an online platform for finding and managing secondhand cars.

## Overview

**MehrCar** is a project developed in the context of **Large-scale and Multi-structured Databases**.

The system is designed as an online application for searching, managing, and publishing secondhand car advertisements. It uses **Java** for the application logic and **MongoDB** as the database technology, making it suitable for handling flexible and multi-structured data. :contentReference[oaicite:1]{index=1}

## Main Idea

The main goal of the project is to provide a platform where users can:

- search for secondhand cars
- manage car-related information and models
- publish advertisements
- browse advertisements using filters such as model and related attributes

According to the current repository description, MehrCar is specifically intended as an application for finding secondhand cars and searching across advertisements based on car data. :contentReference[oaicite:2]{index=2}
